# Gestion des Dépendances

**OBLIGATION STRICTE** : Avant de `commit`, de `push` ou de considérer une tâche comme terminée, si de nouveaux imports ou packages tiers (ex: `date-fns`, `lucide-react`, etc.) ont été introduits dans le code, je DOIS m'assurer de les installer via la commande appropriée (ex: `npm install <package>`). Je dois également vérifier que les fichiers de gestion de paquets (`package.json`, `package-lock.json`, `composer.json`, etc.) ont été mis à jour et les inclure dans les commits.
